# OME Web App #
Built for testing the OME backend. Compatible with all newer devices.
Front End built using Sass, Babel, React, Flux, and many other Node modules.
Bundling by Browserify and orchestrated through Gulp.

## commands ##
```
npm install
npm start
npm start -- -d
```
